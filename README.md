## Steps to deploy a chatbot

1. Get a MessengerX.io API Token from https://portal.messengerx.io 
2. Get  a Wit.ai API Token from https://wit.ai
  - bash chatbot.sh  
       OR
  - go get github.com/dgrijalva/jwt-go  
  - go get github.com/wit-ai/wit-go  
  - go run main.go
